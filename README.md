# videojs-panorama

A [VideoJS][videojs] Fork from videojs-panorama add support for panorama 180 degree videos.


## browser support
IE 11, Edge 12, 13, chrome, firefox, safari, iphone ios 9, ipad ios 9, android 4.4 and above.
